# 🏺 WhimsiCraft

> Handmade clay art, wheel thrown and lovingly glazed in a little studio in Vermont.

A niche artisan clay art website built as a single-page HTML/CSS/JS site — no frameworks, no dependencies, just pure handcrafted code for a handcrafted brand.

## ✨ Features

- Responsive design (mobile, tablet, desktop)
- Sticky nav with cart counter
- Hero split layout with organic clay SVG illustrations
- Scrolling marquee banner
- Product grid with add-to-cart interaction
- Category bento grid
- Studio process section
- Customer testimonials
- Newsletter signup
- Full footer with navigation

## 🎨 Design

- **Fonts:** Playfair Display · Lora · Cormorant Garamond (Google Fonts)
- **Palette:** Terracotta, sage green, warm cream, burnt sienna
- **Aesthetic:** Artisan / organic / cottage-core / niche maker brand
- **Layout:** CSS Grid throughout, no frameworks

## 🛠 Local Development

No build step needed. Just open `index.html` in your browser:

```bash
git clone https://github.com/YOUR_USERNAME/whimsicraft.git
cd whimsicraft
open index.html
```

Or serve locally:

```bash
npx serve .
# or
python3 -m http.server 8080
```

## 📦 Deployment

This site is deployed using **GitHub Pages** from the `main` branch root.

To deploy your own:
1. Fork or clone this repo
2. Go to **Settings → Pages**
3. Set source to **Deploy from branch → main → / (root)**
4. Your site will be live at `https://YOUR_USERNAME.github.io/whimsicraft`
